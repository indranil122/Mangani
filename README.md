# Mangani

A premium, minimalist manga reader built with React, Vite, and Tailwind CSS.

## Features

- **Manga Dex Integration**: Fetches live manga data from the Manga Dex API.
- **Browse Page**: Discover new manga by popularity and category.
- **Search Page**: Search for any manga title.
- **Reader Page**: Premium reading experience with chapter support.
- **Mobile Responsive**: Fully optimized for mobile and desktop screens.

## Tech Stack

- **React**: Modern component-based architecture.
- **Vite**: Ultra-fast build tool.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Lucide React**: For beautiful icons.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server.
